== Description ==

Looking for a clean, simple, and effective way to translate your WordPress site without the bloat?

**Icey - Google Translate** is simply the easiest and best Google Translate plugin for WordPress out there. It replaces the clunky default Google Translate widget with a sleek, customizable modal that fits perfectly into your theme.

From the dedicated **settings page**, you can easily configure the modal's text, set your site's default language, and manage the available translation languages using a smooth drag-and-drop interface. Need to tweak the design? Just drop your custom CSS right into the settings!

== Installation ==

1. Find the plugin in the WordPress repository or upload the plugin folder to `/wp-content/plugins/`.
2. Go to the **Plugins** page in your WordPress admin and activate the plugin.

== Frequently Asked Questions ==

= How do I uninstall the plugin? =
Simply deactivate and delete it from the **Plugins** page.
